# Spotify Clone - Web Application

A responsive Spotify clone web application with music playback functionality, playlist browsing, and user authentication.

## Features

- **Music Player**
  - Play/pause functionality
  - Progress bar with seek control
  - Current track display
  - Time display (current/duration)

- **Playlist Browsing**
  - Recently played tracks
  - Made-for-you playlists
  - Popular playlists
  - Recommended playlists

- **User Authentication**
  - Email/password login
  - Sign up form with validation
  - Social login options (Google, Facebook, Apple)

- **UI Features**
  - Responsive design (mobile & desktop)
  - Dynamic color theme changer
  - Smooth animations and transitions
  - Horizontal scrolling for playlists

## Technologies Used

- HTML5
- CSS3 (with CSS variables)
- JavaScript (ES6)
- SVG for icons and graphics
